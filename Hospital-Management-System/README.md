## 📋 Project Description

A comprehensive **console-based Hospital Management System** developed in C++ for the 1st Semester. This application helps manage hospital operations including patient registration, doctor assignment, room allocation, and patient discharge.

### Features

- ✅ Register new patients with room allocation
- ✅ Search patient by ID
- ✅ Assign available doctors to patients
- ✅ Discharge patients (frees room and doctor)
- ✅ View all admitted patients
- ✅ View list of all doctors with availability status
- ✅ Automatic room and doctor management
- ✅ Data stored using Structures and Arrays
